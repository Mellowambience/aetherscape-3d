# AetherScape 3D

Local-first orthographic **Three.js** OSRS-feel vertical slice in **Ley-Root Hollow** (Aether IP).

## Playable (v0.2 evolved)

- Character create + cloak palettes (Lume / Capkin / Ember / Milo)
- Quest **Relight the Leyward** (gather · combat · craft · return)
- Skills: Combat, Foraging, **Crystal Mining**, Attunement
- Items: Glowreed, Leycord, Shellshard, Hearthloaf, **Raw Crystal**, **Ward Charm**
- **Bank chest**, **day/night**, Shrine Loom recipes (Leycord + Ward Charm)
- Host: `window.AetherScape3DHost` + Eclipse Java glue

## Commands

```bash
npm install
npm test
npm run dev      # http://127.0.0.1:4174/
npm run build && npm run preview
```
